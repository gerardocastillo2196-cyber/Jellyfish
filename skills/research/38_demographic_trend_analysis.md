# Demographic Trend Analysis

**Agencia:** Research
**Rol Sugerido:** @market_researcher

## Objetivo de la Skill
Interpreta piramides poblacionales, datos demograficos, y trends de mercado para informar decisiones de producto y negocio.

## Metodologia de Razonamiento (Paso a Paso)
1. **Data collection**:
   - Census data
   - UN/DO population projections
   - Regional statistical offices
2. **Pyramid analysis**:
   - Age distribution patterns
   - Dependency ratios
   - Gender ratios
   - Cohort analysis
3. **Trend identification**:
   - Aging population indicators
   - Urbanization trends
   - Household composition changes
4. **Life stage analysis**:
   - Young singles/couples
   - Growing families
   - Empty nesters
   - Retirees
5. **Market segmentation**:
   - TAM/SAM/SOM based on demographics
6. **Geographic implications**:
   - Rural vs urban trends
   - Regional variations

## Anti-Patrones
- Using outdated census data.
- Ignoring migration patterns.
- Treating all "Boomers" or "Millennials" the same.
- Not considering cohort effects.
- Over-generalizing from national data.
